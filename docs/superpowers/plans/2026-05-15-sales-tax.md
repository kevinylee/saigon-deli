# Sales Tax Collection Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Charge Seattle, WA 10.55% sales tax on every Stripe Checkout order. Tax applies to food only (tip is not taxed) and is computed before tip in display order.

**Architecture:** Server-side, attach a Stripe `TaxRate` object to each food line item in `checkout.ts`; tip line item omits the rate, so Stripe excludes it. Client-side, mirror Stripe's per-line rounding in `CheckoutModal.jsx` for an accurate preview. Persist a synthetic Sales Tax row into `array_line_items` at webhook time so the admin dashboard's existing line-item rendering works without a DB migration. Surface tax to the customer receipt via a separate API field.

**Tech Stack:** Gatsby (React), Netlify Functions (TypeScript), Stripe Checkout, Supabase. No automated test harness — verification is manual via `netlify dev` + Stripe test mode.

**Spec:** [docs/superpowers/specs/2026-05-15-sales-tax-design.md](../specs/2026-05-15-sales-tax-design.md)

**Files:**

The receipt page renders Tax outside the line items array (in `<tfoot>`). The API needs to expose it as its own field. Both code paths (Supabase hit and Stripe fallback) must populate it.

- [ ] **Step 1: Update getReceiptData to return tax**

Open `netlify/functions/receipt.ts`. Replace the entire `getReceiptData` function (currently lines 44–61) with:

```ts
const getReceiptData = async (sessionId: string) => {
  const { data } = await supabase
    .from('Orders')
    .select('*')
    .eq('session_id', sessionId)
    .single()

  if (data) {
    const taxRow = data.array_line_items.find((li: any) => li.title === 'Sales Tax');
    const tax = taxRow?.amount_total ?? 0;
    const itemsOnly = data.array_line_items.filter((li: any) => li.title !== 'Sales Tax');
    const normalizedLineItems = itemsOnly.map(normalizeLineItem);
    return [normalizedLineItems, data.pickup_at, tax];
  } else {
    const checkoutSession = await stripe.checkout.sessions.retrieve(sessionId)
    const lineItemPayload = await stripe.checkout.sessions.listLineItems(sessionId, { limit: 100, expand: ["data.price.product"] })

    const normalizedLineItems = lineItemPayload.data.map(normalizeLineItem);
    const tax = checkoutSession.total_details?.amount_tax ?? 0;
    return [normalizedLineItems, checkoutSession.metadata?.pickup_time, tax]
  }
}
```

3. Update `GATSBY_TAX_RATE_PERCENT` in Netlify env to match the new percentage.
